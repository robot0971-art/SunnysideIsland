## 게임 소개

**써니사이드 아일랜드**는 낡은 섬을 관광지로 개발하는 생존/경영 시뮬레이션 게임입니다. 28일 동안 섬을 발전시키고, 관광객을 유치하며, 궁극적으로 섬의 번영을 이끌어내세요.

### 게임 특징

- 🏝️ **섬 개발**: 다양한 건물을 건설하여 섬을 발전시키세요
- 🎣 **생활 콘텐츠**: 농사, 낚시, 채집으로 자원을 획득하세요
- 👥 **주민 고용**: 주민을 고용하여 건물을 운영하세요
- 🏛️ **관광 산업**: 관광객을 유치하여 수익을 창출하세요
- ⚔️ **전투**: 고블린을 물리치고 섬을 지키세요
- 📜 **퀘스트**: 메인 퀘스트를 통해 스토리를 진행하세요

## 설치 방법

### Windows

1. 설치 파일(`SunnysideIsland_Setup.exe`)을 실행합니다
2. 설치 경로를 선택합니다
3. '설치' 버튼을 클릭합니다
4. 설치 완료 후 '게임 실행'을 체크하고 '완료'를 클릭합니다

### 포터블 버전

1. 압축 파일을 원하는 위치에 해제합니다
2. `SunnysideIsland.exe`를 실행합니다

## 게임 시작하기

### 새 게임

1. 타이틀 화면에서 '새 게임'을 선택합니다
2. 튜토리얼이 진행됩니다 (스킵 가능)
3. Day 1 아침 6시부터 게임이 시작됩니다

### 불러오기

1. 타이틀 화면에서 '불러오기'를 선택합니다
2. 저장 슬롯을 선택합니다
3. 자동 저장과 수동 저장이 모두 표시됩니다

## 게임 시스템

### 생존 시스템

#### 체력 (Health)
- 적에게 공격받으면 감소합니다
- 0이 되면 사망합니다
- 음식 섭취나 수면으로 회복합니다

#### 허기 (Hunger)
- 시간이 지나면 감소합니다
- 0에 가까워질수록 체력이 감소합니다
- 음식을 섭취하여 회복합니다

#### 스태미나 (Stamina)
- 달리기, 구르기, 공격 시 소모됩니다
- 가만히 있으면 자연 회복됩니다
- 0이면 달리기와 구르기가 불가능합니다

### 시간 시스템

- **1초(실제) = 1분(게임)**
- **하루 = 24분(실제)**
- 시간대: 새벽 → 아침 → 정오 → 오후 → 저녁 → 밤

### 계절 시스템

| 계절 | 일수 |
|------|------|
| 봄 | Day 1-7 |
| 여름 | Day 8-14 |
| 가을 | Day 15-21 |
| 겨울 | Day 22-28 |

### 날씨 시스템

| 날씨 | 효과 |
|------|------|
| 맑음 | 기본 상태 |
| 흐림 | 관광객 감소 |
| 비 | 농사 성장 증가, 낚시 확률 증가 |
| 폭풍 | 야외 활동 제한 |
| 무지개 | 행운 증가 |

## 퀘스트 가이드

### 챕터 구성

#### Chapter 1: 새로운 시작 (Day 1-3)
- 섬 적응 및 기본 시스템 학습
- 주요 퀘스트: 기본 자원 확보, 첫 건설

- 건설 및 경제 시스템 본격 시작

#### Chapter 3: 번영의 길 (Day 8-14)

- 주요 퀘스트: 관광 시설 건설, 평판 향상

#### Chapter 4: 전설의 섬 (Day 15-28)

## 팁과 전략

### 초반 팁 (Day 1-7)

1. **튜토리얼을 따르세요** - 게임의 기본을 배울 수 있습니다
2. **자원을 모으세요** - 나무, 돌, 약초는 항상 필요합니다
3. **밭을 일구세요** - 안정적인 식량 공급이 중요합니다
4. **텐트를 업그레이드하세요** - 주민을 고용하려면 주거 공간이 필요합니다

### 중반 팁 (Day 8-14)

1. **관광 시설에 투자하세요** - 수익의 주된 원천입니다
2. **주민을 적절히 배치하세요** - 건물마다 적합한 주민이 있습니다
3. **날씨를 확인하세요** - 농사와 낚시에 영향을 줍니다
4. **서브 퀘스트를 수행하세요** - 추가 보상을 얻을 수 있습니다

### 후반 팁 (Day 15-28)

1. **방어 시설을 강화하세요** - 고블린 공격이 빈번해집니다
2. **장비를 업그레이드하세요** - 보스전에 대비하세요
3. **평판을 관리하세요** - 더 많은 관광객을 유치할 수 있습니다
4. **자원을 비축하세요** - 겨울에는 농사가 어렵습니다

## 문제 해결

### 자주 묻는 질문

**Q: 게임이 실행되지 않아요**
- 시스템 요구사항을 확인하세요
- DirectX가 설치되어 있는지 확인하세요
- 바이러스 백신에서 게임을 제외 목록에 추가하세요

**Q: 저장이 되지 않아요**
- 저장 공간이 충분한지 확인하세요
- 관리자 권한으로 실행해 보세요
- `%USERPROFILE%\AppData\Local\SunnysideIsland\Saves\` 폴더 권한을 확인하세요

**Q: 프레임이 떨어져요**
- 그래픽 설정을 낮추세요
- 백그라운드 프로그램을 종료하세요
- 최신 그래픽 드라이버를 설치하세요

### 기술 지원

문제가 지속되면 다음으로 연락하세요:
