## Version 1.0.0 (Release)

**출시일**: 2026-03-XX

### 새로운 기능

#### 게임 시스템
- ✨ 기본 생존 시스템 (체력, 허기, 스태미나)
- ✨ 시간/계절/날씨 시스템
- ✨ 인벤토리 및 아이템 시스템
- ✨ 저장/불러오기 시스템

#### 생활 콘텐츠
- ✨ 농사 시스템 (심기, 물주기, 수확)
- ✨ 낚시 시스템 (미니게임)
- ✨ 채집 시스템 (나무, 광석, 약초)

#### 건설 시스템
- ✨ 30+ 건물 건설 가능
- ✨ 5개 카테고리 (주거, 상업, 관광, 생산, 방어)
- ✨ 건물 업그레이드 시스템

#### 경제 시스템
- ✨ 통화(골드) 시스템
- ✨ 상점 구매/판매
- ✨ 관광객 시스템
- ✨ 주민 고용 시스템

#### 전투 시스템
- ✨ 근접 무기 전투
- ✨ 적 AI (고블린, 고블린 족장)
- ✨ 보스전 (고블린 족장)

#### 콘텐츠
- ✨ 28일 메인 스토리

- ✨ 18개 서브 퀘스트

- 일부 환경에서 전체화면 전환 시 깜빡임 발생 가능
