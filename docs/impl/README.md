# Sunnyside Island 구현 계획

## 개요
이 폴더는 Sunnyside Island 게임의 기술적 구현 계획 문서들을 포함합니다.

## 🚀 개발 시작하기

### 1. 반드시 먼저 읽어야 할 문서
- **[WORKFLOW.md](./WORKFLOW.md)** - 개발 프로세스 및 규칙
- **[CHECKLIST.md](./CHECKLIST.md)** - 마일스톤별 작업 체크리스트
- **[AGENTS.md](../../AGENTS.md)** - 코딩 표준

### 2. 개발 프로세스
```
1. CHECKLIST.md에서 현재 Phase 확인
2. phases/M{X}/ 폴더에 RESEARCH 문서 작성
3. PLAN 문서 작성
4. 구현
5. 체크리스트 업데이트
6. 다음 작업으로 이동
```

## 📋 문서 목록

### 개발 프로세스
- **[WORKFLOW.md](./WORKFLOW.md)** - Phase 기반 개발 프로세스 및 규칙
- **[CHECKLIST.md](./CHECKLIST.md)** - 마일스톤별 상세 체크리스트 및 진행 상황

### Phase별 작업 문서 (phases/)
각 마일스톤별로 리서치 및 계획 문서를 관리합니다.

```
phases/
├── M0/                    # 프로젝트 설정
├── M1/                    # 코어 시스템
│   ├── RESEARCH_M1_DI.md
│   ├── PLAN_M1_DI.md
│   └── ...
├── M2/                    # 생존/생산 시스템
├── M3/                    # 전투/건설 시스템
├── M4/                    # 경제/콘텐츠 시스템
├── M5/                    # 콘텐츠 완성
├── M6/                    # 폴리싱
└── M7/                    # 릴리즈
```

### 5. [UI/UX 시스템](05-ui-ux.md)

### 7. [테스트 계획](07-testing.md)

## 우선순위 요약

### P0 (필수)
- 코어 시스템 (DI, EventBus, 저장)
- 플레이어 기본 컨트롤
- 생존 시스템 (허기, 체력, 스태미나)
- 인벤토리 시스템
- 시간 관리 시스템
- 기본 UI (HUD, 인벤토리)

### P1 (권장)
- 농사 시스템
- 낚시 시스템
- 건설 시스템
- 전투 시스템
- 상점 시스템
- 퀘스트 시스템

### P2 (선택)
- 이벤트 시스템
- 업적 시스템
- 특수 효과
- 추가 UI 스킨
