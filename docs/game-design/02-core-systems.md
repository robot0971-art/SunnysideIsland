# 2. 핵심 시스템 상세

## 2.1 시간 시스템

### 시간 흐름
- **실시간**: 1초 = 1분 (게임 내)
- **하루**: 24분 (실제 시간)
- **게임 내 1개월**: 28일 (4주)
- **총 게임 기간**: 28일 (약 11시간 실시간, 타임랩스 기능으로 3시간 플레이)

### 시간대 효과

| 시간대 | 게임 시간 | 특징 | 활동 추천 |
|--------|-----------|------|-----------|
| 새벽 | 04:00-06:00 | 어두움, 몬스터 활동 ↑ | - |
| 아침 | 06:00-09:00 | 밝아짐, NPC 기상 | 농사, 채집 |
| 오전 | 09:00-12:00 | 활동 최적기 | 건설, 탐험 |
| 점심 | 12:00-14:00 | NPC 휴식 시간 | 낚시, 사냥 |
| 오후 | 14:00-18:00 | 황금 시간 | 전투, 채집 |
| 저녁 | 18:00-21:00 | 어두워짐 | 상점 운영, 요리 |
| 밤 | 21:00-04:00 | 어두움, 몬스터 활동 최대 | 휴식, 동굴 탐험 |

### 날씨 시스템

| 날씨 | 확률 | 효과 |
|------|------|------|
| 맑음 | 50% | 기본 상태, 모든 활동 가능 |
| 흐림 | 20% | 낚시 성공률 +10% |
| 비 | 15% | 농사 자동 물주기, 채집 불편 |
| 폭풍 | 10% | 외부 활동 위험, 실내 권장 |
| 무지개 | 5% | 희귀, 행운 +20, 관광객 증가 |

## 2.2 생존 시스템

### 허기 (Hunger)

```
기본 수치: 100/100
감소율: 1.5/분 (게임 내)
하루 소모: 약 216 (실제로는 음식으로 충당)
```

| 허기 상태 | 수치 | 효과 |
|-----------|------|------|
| 포만 | 80-100 | 체력 회복 +0.5/초 |
| 보통 | 40-79 | 정상 |
| 배고픔 | 20-39 | 이동 속도 -10% |
| 굶주림 | 0-19 | 체력 -0.5/초, 이동 속도 -30% |

### 체력 (Health)

```
기본 수치: 100/100
회복: 허기 80+ 일 때 0.5/초
사망: 0이 되면 최근 저장 지점에서 부활
```

### 스태미나 (Stamina)

```
기본 수치: 100/100
소모: 공격, 구르기, 빠른 이동
회복: 서 있을 때 5/초
```

## 2.3 인벤토리 시스템

### 인벤토리 구조

```
메인 인벤토리: 24칸 (4x6)
퀵슬롯: 8칸 (1-8 키)
장비 슬롯: 무기, 갑옷, 액세서리
돈: 코인 (별도 UI로 표시)
```

### 아이템 분류

- **도구**: 내구도 있음, 수리 가능
- **소비**: 음식, 물약
- **재료**: 조합/건설용
- **귀중품**: 판매용
- **퀘스트**: 진행용

```

## 2.5 낚시 시스템

### 낚시 메커닉

```
1. 워터 타일 근처에서 'F'키로 낚시 시작
2. 대기 (물고기가 미끼를 무는 시간: 3-10초)
3. 물결 표시 시 'F'키 연타
4. 미터 게이지가 초록색 영역일 때 'F'키
5. 체력이 0이 되면 성공!
```

### 물고기 도감

| 물고기 | 등급 | 출현 장소 | 시간 | 난이도 | 판매가 |
|--------|------|-----------|------|--------|--------|
| 농어 | ★ | 바다 | 전 시간 | 쉬움 | 30 |
| 잉어 | ★ | 호수 | 전 시간 | 쉬움 | 25 |
| 메기 | ★ | 강 | 밤 | 쉬움 | 35 |
| 참치 | ★★ | 바다 깊은 곳 | 낮 | 보통 | 80 |
| 연어 | ★★ | 폭포 | 아침 | 보통 | 100 |
| 황새치 | ★★★ | 바다 | 낮 | 어려움 | 250 |
| 상어 | ★★★ | 바다 깊은 곳 | 밤 | 매우 어려움 | 500 |
| 황금 물고기 | ★★★★ | 특별 장소 | 무지개 날 | 극악 | 2000 |

### 낚싯대 종류

| 낚싯대 | 성공률 | 특징 |
|--------|--------|------|
| 나무 낚싯대 | 표준 | 기본 |
| 철제 낚싯대 | +20% | 희귀 물고기 확률 ↑ |
| 금 낚싯대 | +35% | 희귀 물고기 확률 ↑ |
| 전설의 낚싯대 | +50% | 황금 물고기 출현 |

| 토끼 | 20 | 빠름 | 고기(1), 가죽(1) | 도망형 |
| 사슴 | 40 | 매우 빠름 | 고기(2), 가죽(2) | 도망형 |

### 사냥 방법

**근접 사냥**
- 은신 접근: 풀 속에서 Shift 키로 웅크리기

**원거리 사냥**
- 활 장착 후 마우스 우클릭 조준
- 게이지 차는 동안 홀드 (풀면 발사)
- 풀 차지 시 데미지 x2

## 2.10 관광객 시스템

### 관광객 메커닉

```
1. 부두에 배 도착 (오전 09:00 고정)
2. 관광객이 마을 탐방 시작
3. 관광객 AI가 시설을 찾아다니며 소비
4. 저녁 6시부터 숙소로 이동
5. 다음날 아침 9시 배로 귀환
