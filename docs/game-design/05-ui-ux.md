### HUD 요소 상세

#### 체력 바

```
위치: 좌측 상단
크기: 200x20px
디자인: 빨간색 하트 아이콘 + 막대 그래프
표시: ♥ x 10 또는 연속된 하트 아이콘
색상: 100% 초록 → 50% 노랑 → 25% 빨강
```

#### 허기 바

```
위치: 체력 바 우측
크기: 150x20px
디자인: 고기 아이콘 + 막대 그래프
표시: 🍖 x 5 또는 연속된 고기 아이콘
색상: 100% 주황 → 50% 노랑 → 25% 빨강
```

#### 코인

```
위치: 우측 상단
디자인: ⓒ + 숫자
색상: 금색
폰트: 볼드, 24pt
```

#### 시간/날짜

```
위치: 코인 하단
형식: Day 5 14:30
디자인: 시계 아이콘 + 텍스트
색상: 흰색
```

#### 날씨/평판

```
위치: 시간 하단
날씨: 아이콘 (☀️🌧️⛈️🌈)
평판: 별 5개 중 색칠된 개수 (⭐⭐⭐☆☆)
```

## 5.3 피드백 시스템

### 시각적 피드백

#### 타격 피드백

```
- 적 피격 시: 흰색 깜빡임 (0.2초)
- 데미지 숫자: 떠오르는 텍스트
  - 일반: 흰색
  - 치명타: 빨간색 + 크게
  - 약점: 노란색
- 피격 방향: 넉백 애니메이션
```

#### 수집 피드백

```
- 아이템 획득: 아이템 아이콘이 위로 떠오름
- 효과: 반짝임 + 사운드
- 텍스트: "+ [아이템명]" 표시 (2초)
```

#### 레벨업 피드백

```
- 화면 중앙: "LEVEL UP!" 텍스트
- 황금 빛 파티클 효과
- 레벨업 사운드
- 스킬 포인트 +1 표시
```

#### 퀘스트 피드백

```
완료 시:
- 화면 우측 상단: 알림 창
- 제목: 퀘스트명
- 보상: 아이콘 + 수량
- 체크 표시 애니메이션
- 자동 사라짐 (5초)
```

#### 낮/밤 전환

```
- 전환 시간: 3초
- 화면 점점 어두워짐/밝아짐
- BGM 전환
- NPC 행동 변화
```

### 사운드 피드백

#### 발소리

| 표면 | 소리 |
|------|------|
| 흙 | 부드러운 발소리 |
| 나무 | 딱딱한 발소리 |
| 돌 | 단단한 발소리 |
| 물 | 첨벙거리는 소리 |
| 잔디 | 바스락 소리 |

#### 타격음

| 무기 | 소리 |
|------|------|
| 검 | 쇳소리 + 바람소리 |
| 도끼 | 묵직한 타격음 |
| 활 | 쉭~ 퍽 |
| 맨손 | 퍽퍽 |

#### 환경음

```
- 바람: 바스락 (풀, 나무)
- 파도: 밀려오는 소리 (해변)
- 새소리: 지저귀는 소리 (숲)
- 벌레: 울음소리 (밤)
```

### BGM 테마

| 시간/상황 | 음악 스타일 |
|-----------|-------------|
| 아침 | 경쾌한, 희망적인 |
| 낮 | 여유로운, 평화로운 |
| 저녁 | 잔잔한, 따뜻한 |
| 밤 | 조용한, 긴장감 있는 |
| 전투 | 빠르고 긴장된 |
| 보스전 | 웅장하고 어두운 |
| 축제 | 신나는, 즐거운 |

## 5.4 헬퍼 시스템

### 튜토리얼

#### Day 1 튜토리얼 흐름

**1. 이동 튜토리얼**
```
화면: "WASD로 이동해보세요"
조건: 5칸 이동
완료: 체크 표시
```

**2. 채집 튜토리얼**
```
화면: "나무를 향해 가서 F를 눌러 베어보세요"
조건: 나무 1개 벌목
완료: 아이템 획득 애니메이션
```

**3. 인벤토리 튜토리얼**
```
화면: "I 키를 눌러 인벤토리를 열어보세요"
조건: 인벤토리 열기
완료: 인벤토리 설명 UI
```

**4. 건설 튜토리얼**
```
화면: "B 키로 건설 메뉴를 열고 텐트를 지으세요"
조건: 텐트 건설
완료: 건설 완성 애니메이션
```

**5. 생존 튜토리얼**
```
화면: "허기가 떨어지면 낚시하거나 사냥하세요"
조건: 음식 섭취
완료: 허기 회복 표시
```

## 5.6 반응형 UI

### 화면 비율 대응

```
기준: 1920x1080 (16:9)
지원: 16:9, 16:10, 4:3, 21:9

적용 방식:
- HUD: 화면 기준 위치 (%)
- 메뉴: 중앙 정렬, 최대 크기 제한
- 월드: Pixel Perfect Camera 유지
```

### 해상도 스케일링

| 해상도 | UI 스케일 |
|--------|-----------|
| 1280x720 | 0.75x |
| 1920x1080 | 1.0x (기준) |
| 2560x1440 | 1.33x |
| 3840x2160 | 2.0x |

### 접근성 옵션

```
- 글자 크기 조절 (0.8x ~ 1.5x)
- UI 투명도 조절
- 색맹 모드 (적/녹색 대체)
- 자동 진행 속도 조절
```
