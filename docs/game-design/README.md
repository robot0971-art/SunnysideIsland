# 선샤인 아일랜드 (Sunnyside Island) - GDD 문서 모음

## 게임 개요
버려진 섬에 불시착한 주인공이 자원을 채집하고, 농사와 낚시, 사냥으로 생존하며, 점차 섬을 관광 명소로 발전시키는 생존 건설 시뮬레이션
