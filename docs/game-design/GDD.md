## 1. 게임 개요

### 1.1 기본 정보
| 항목 | 내용 |
|------|------|
| 게임명 | 선샤인 아일랜드 (Sunnyside Island) |
| 장르 | 생존 / 건설 / 팜딩 / 액션 RPG |
| 타겟 플랫폼 | PC (Windows/Mac) |
| 엔진 | Unity 2022.3 LTS |
| 플레이 시간 | 2.5 ~ 3.5시간 |
| 플레이어 수 | 1인 (싱글플레이어) |
| 시점 | 탑다운 2D (16x16 픽셀 아트) |
| ESRB 등급 | E10+ (10세 이상) |

### 1.2 한 줄 컨셉
버려진 섬에 불시착한 주인공이 자원을 채집하고, 농사와 낚시, 사냥으로 생존하며, 점차 섬을 관광 명소로 발전시키는 생존 건설 시뮬레이션

### 1.3 핵심 가치
- **성취감**: 텐트 하나에서 시작해 번화한 리조트로 발전
- **자유도**: 플레이어의 선택에 따른 다양한 발전 방식
- **힐링**: 귀여운 픽셀 아트와 여유로운 게임플레이
- **도전**: 전투와 생존의 긴장감

### 1.4 플레이어 여정
```
[Day 1-3]  생존자     → 텐트, 낚시, 기본 도구
[Day 4-7]  정착민     → 오두막, 농사, 첫 주민
[Day 8-14] 마을 주민  → 상점, 고블린 교역, 관광객
[Day 15-28] 리조트 주인 → 고급 시설, 축제, 번영
```

### 1.5 게임 흐름 다이어그램
```
자원 채집 (벌목/채광/사냥/낚시)
        ↓
생존 유지 (허기 관리/요리)
        ↓
건설/확장 (농장/상점/주거지)
        ↓
주민/관광객 유치
        ↓
수익 창출 → 더 큰 시설
        ↓
리조트 완성 (엔딩)
```

## 2. 핵심 시스템 상세

### 2.1 시간 시스템

#### 시간 흐름
- **실시간**: 1초 = 1분 (게임 내)
- **하루**: 24분 (실제 시간)
- **게임 내 1개월**: 28일 (4주)
- **총 게임 기간**: 28일 (약 11시간 실시간, 타임랩스 기능으로 3시간 플레이)

#### 시간대 효과
| 시간대 | 게임 시간 | 특징 | 활동 추천 |
|--------|-----------|------|-----------|
| 새벽 | 04:00-06:00 | 어두움, 몬스터 활동 ↑ | - |
| 아침 | 06:00-09:00 | 밝아짐, NPC 기상 | 농사, 채집 |
| 오전 | 09:00-12:00 | 활동 최적기 | 건설, 탐험 |
| 점심 | 12:00-14:00 | NPC 휴식 시간 | 낚시, 사냥 |
| 오후 | 14:00-18:00 | 황금 시간 | 전투, 채집 |
| 저녁 | 18:00-21:00 | 어두워짐 | 상점 운영, 요리 |
| 밤 | 21:00-04:00 | 어두움, 몬스터 활동 최대 | 휴식, 동굴 탐험 |

#### 날씨 시스템
- **맑음 (50%)**: 기본 상태, 모든 활동 가능
- **흐림 (20%)**: 낚시 성공률 +10%
- **비 (15%)**: 농사 자동 물주기, 채집 불편
- **폭풍 (10%)**: 외부 활동 위험, 실내 권장
- **무지개 (5%)**: 희귀, 행운 +20, 관광객 증가

### 2.2 생존 시스템

#### 허기 (Hunger)
```
기본 수치: 100/100
감소율: 1.5/분 (게임 내)
하루 소모: 약 216 (실제로는 음식으로 충당)
```

| 허기 상태 | 수치 | 효과 |
|-----------|------|------|
| 포만 | 80-100 | 체력 회복 +0.5/초 |
| 보통 | 40-79 | 정상 |
| 배고픔 | 20-39 | 이동 속도 -10% |
| 굶주림 | 0-19 | 체력 -0.5/초, 이동 속도 -30% |

#### 체력 (Health)
```
기본 수치: 100/100
회복: 허기 80+ 일 때 0.5/초
사망: 0이 되면 최근 저장 지점에서 부활
```

#### 스태미나 (Stamina)
```
기본 수치: 100/100
소모: 공격, 구르기, 빠른 이동
회복: 서 있을 때 5/초
```

#### 피로도 (Fatigue) - 선택적
```
하루 종일 활동 시 누적
취침 시 초기화
텐트: 피로도 100% → 0%
침대: 피로도 100% → 0% + 체력 +20
```

#### 인벤토리 구조

- **소비**: 음식, 물약
- **재료**: 조합/건설용
- **귀중품**: 판매용

#### 무게 시스템 (선택적)
```
캐릭터 무게 제한: 100kg
초과 시: 이동 속도 감소
저장고/상자: 무제한
```

### 2.5 낚시 시스템 상세

#### 낚시 메커닉
```
1. 워터 타일 근처에서 'F'키로 낚시 시작
2. 대기 (물고기가 미끼를 무는 시간: 3-10초)
3. 물결 표시 시 'F'키 연타
4. 미터 게이지가 초록색 영역일 때 'F'키
5. 체력이 0이 되면 성공!
```

#### 낚싯대 종류

#### 사냥 방법

**근접 사냥**
```
- 은신 접근: 풀 속에서 Shift 키로 웅크리기
- 토끼/사슴: 3초 이내 접근 시 기습 성공 (데미지 x2)
- 공격: 마우스 좌클릭 또는 스페이스바
- 콤보: 3연속 공격 시 마지막 데미지 +50%
```

**원거리 사냥**
```
- 활 장착 후 마우스 우클릭 조준
- 게이지 차는 동안 홀드 (풀면 발사)
- 풀 차지 시 데미지 x2, 정확도 100%
- 화살은 제작하거나 상점에서 구매
```

#### 도망 AI 패턴
- **토끼**: 직선 도망, 3초 후 은신 시도
- **사슴**: 지그재그 도망, 무리 행동
- **멧돼지**: 충격 후 도망 또는 반격
