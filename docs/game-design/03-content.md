# 3. 콘텐츠 상세

## 3.1 조합(크래프팅) 시스템

### 조합 카테고리

- **도구**: 채집 도구, 전투 도구
- **장비**: 갑옷, 장신구
- **가구**: 집 꾸미기
- **건축**: 건설 재료
- **요리**: 음식
- **가공**: 원재료 가공

### 주요 조합 레시피

#### 도구

```
나무 도끼: 나무 x10
돌 곡괭이: 나무 x5 + 돌 x10
철 낚싯대: 나무 x10 + 철 x5
철 검: 철 x10 + 나무 x5
활: 나무 x15 + 줄 x5
```

#### 건축 재료

```
나무 판자: 나무 x2
돌 벽돌: 돌 x2
철 주괴: 철광석 x3 (화덕에서)
강철: 철 주괴 x2 + 석탄 x1
```

#### 가구 (집 꾸미기)

```
침대: 나무 x20 + 천 x10
책상: 나무 x15
의자: 나무 x10
선반: 나무 x10
벽난로: 돌 x30
카펫: 천 x5
```

## 3.2 요리 시스템

### 요리 메커닉

```
1. 화덕 또는 부엌 근처에서 상호작용
2. 레시피 선택 또는 재료 조합
3. 조리 시간 대기
4. 완성된 요리 수령
```

### 요리 목록

| 요리 | 재료 | 허기 회복 | 추가 효과 | 조리 시간 |
|------|------|-----------|-----------|-----------|
| 구운 생선 | 생선 x1 | +20 | 없음 | 30초 |
| 생선구이 | 생선 x2 + 소금 x1 | +45 | 체력 +5 | 1분 |
| 스튜 | 고기 x2 + 감자 x2 + 당근 x1 | +60 | 체력 +10 | 2분 |
| 샐러드 | 채소 x3 | +30 | 스태미나 +20 | 30초 |
| 빵 | 밀가루 x2 | +40 | 없음 | 3분 |
| 샌드위치 | 빵 x2 + 고기 x1 + 채소 x1 | +50 | 없음 | 1분 |
| 파이 | 밀가루 x2 + 과일 x3 | +55 | 기분 상승 | 4분 |
| 스테이크 | 고급 고기 x2 | +80 | 공격력 +10% (10분) | 2분 |
| 해산물 파스타 | 생선 x2 + 밀가루 x2 | +70 | 모든 능력치 +5 | 3분 |
| 로얄 플래터 | 생선 x3 + 고기 x3 + 채소 x3 | +100 | 모든 능력치 +15 (30분) | 5분 |

### 특별 요리 (레어)

**황금 물고기 요리**
- 재료: 황금 물고기 x1
- 효과: 허기 +100, 24시간 행운 최대, 체력 자동 회복

## 3.3 퀘스트 시스템

### 퀘스트 유형

- **메인 퀘스트**: 스토리 진행
- **서브 퀘스트**: NPC 요청
- **일일 퀘스트**: 반복 가능
- **업적**: 도전 과제

### 메인 퀘스트 라인

#### Chapter 1: 생존 (Day 1-3)

**[불시착]**
- 목표: 텐트 건설
- 보상: 기본 도구 세트

**[첫 식사]**
- 목표: 낚시로 생선 3마리 잡기
- 보상: 낚싯대 내구도 증가

**[안전한 곳]**
- 목표: 오두막 완성
- 보상: 침대 제작법

#### Chapter 2: 정착 (Day 4-7)

**[농부의 시작]**
- 목표: 첫 작물 수확
- 보상: 씨앗 20개

**[새로운 이웃]**
- 목표: 첫 주민 유치
- 보상: 주민 맞춤 가구 세트

**[고블린의 위협]**
- 목표: 고블린 5마리 처치
- 보상: 전투 도구 제작법

#### Chapter 3: 교류 (Day 8-14)

**[고블린 마을 발견]**
- 목표: 고블린 마을 탐색
- 보상: 고블린어 번역기

**[평화의 조건]**
- 선택 A: 고블린 족장 처치 (전투)
- 선택 B: 교역 협정 체결 (외교)
- 보상: 고블린 동맹 or 고블린 보물

**[첫 관광객]**
- 목표: 관광객 10명 유치
- 보상: 부두 업그레이드

#### Chapter 4: 번영 (Day 15-28)

**[상인의 길]**
- 목표: 상점 5개 건설
- 보상: 시장 건설 가능

**[축제의 날]**
- 목표: 첫 축제 개최
- 보상: 관광객 평판 대폭 상승

**[리조트 완성]**
- 목표: 리조트 호텔 건설
- 보상: 게임 클리어, 자유 모드 해금

### NPC 서브 퀘스트 예시

| NPC | 퀘스트 | 목표 | 보상 |
|-----|--------|------|------|
| 낚시꾼 존 | 전설의 물고기 | 황금 물고기 낚시 | 전설 낚싯대 |
| 대장장이 | 희귀 광석 | 미스릴 10개 채굴 | 강철 갑옷 |

| 고블린 상인 | 물물교환 | 인간 물건 10개 | 고블린 특산물 |
| 탐험가 | 잃어버린 유물 | 고대 유적 탐색 | 지도 및 보물 |

## 3.4 스킬/성장 시스템

### 스킬 트리

플레이어는 활동을 통해 경험치 획득, 레벨 업 시 스킬 포인트 획득

#### 채집 계열

```
Lv1: 벌목 속도 +10%
Lv2: 채광 속도 +10%
Lv3: 희귀 자원 발견 +5%
Lv4: 내구도 소모 -20%
Lv5: 한 번에 2개 채집 가능
```

#### 농사 계열

```
Lv1: 성장 속도 +10%
Lv2: 수확량 +1
Lv3: 물 주기 필요 주기 -1일
Lv4: 품질 상승 확률 +20%
Lv5: 계절 무관 작물 재배
```

#### 낚시 계열

```
Lv1: 미끼 대기 시간 -20%
Lv2: 미니게임 난이도 -10%
Lv3: 희귀 물고기 확률 +10%
Lv4: 낚싯대 내구도 소모 -30%
Lv5: 물고기 크기/품질 상승
```

#### 전투 계열

```
Lv1: 공격력 +10%
Lv2: 방어력 +10%
Lv3: 치명타 확률 +5%
Lv4: 스태미나 소모 -20%
Lv5: 특수 공격 해금 (회전베기)
```

#### 건설/경제 계열

```
Lv1: 건설 속도 +20%
Lv2: 건설 비용 -10%
Lv3: 판매 가격 +10%
Lv4: 주민 생산성 +15%
Lv5: 관광객 유치 +25%
```

### 레벨 및 경험치

```
레벨 1 → 2: 100 XP
레벨 2 → 3: 200 XP
레벨 3 → 4: 400 XP
...
레벨 10: 약 50,000 XP 누적

활동별 XP:
- 벌목: 5 XP
- 채광: 10 XP
- 작물 수확: 20 XP
- 물고기 낚시: 15 XP
- 몬스터 처치: 30 XP
- 건설 완료: 50 XP
- 퀘스트 완료: 100~500 XP
```

## 3.5 특별 이벤트

### 정기 이벤트

**주간 이벤트**

| 요일 | 이벤트 | 효과 |
|------|--------|------|
| 월요일 | 채집의 날 | 채집 XP 2배 |
| 수요일 | 낚시 대회 | 특별 물고기 출현 |
| 금요일 | 시장의 날 | 모든 가격 할인 20% |
| 일요일 | 쉬는 날 | 모든 NPC 휴식 |

**계절 이벤트**

| 계절 | 이벤트 | 효과 |
|------|--------|------|
| 봄 | 식목일 | 나무 무료 제공 |
| 여름 | 해변 축제 | 관광객 2배 |
| 가을 | 추수제 | 농작물 수확량 2배 |
| 겨울 | 눈축제 | 겨울 한정 아이템 |

### 랜덤 이벤트

| 이벤트 | 확률 | 효과 |
|--------|------|------|
| 상인 방문 | 10%/일 | 희귀 아이템 판매 (1일) |
| 고블린 습격 | 5%/일 | 마을 방어전 |
| 보물 발견 | 2%/일 | 랜덤 보물 상자 |
| 무지개 | 1%/일 | 행운 최대 (1일) |
| 유령 출몰 | 3%/밤 | 특별 몬스터 출현 |
| 풍작 | 5%/일 | 작물 성장 속도 2배 (3일) |
| 가뭄 | 3%/일 | 물 주기 필요 증가 (3일) |
