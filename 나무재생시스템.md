# 나무 재생 시스템 (Tree Regeneration System) 설계서

이 문서는 별도의 그루터기(Stump) 에셋 없이, **스케일(Scale) 조절**과 **이벤트 시스템**을 활용하여 벌목된 나무가 매일 조금씩 자라나 다시 벨 수 있는 상태가 되는 시스템을 정의합니다.

---

## 1. 시스템 요약
- **성장 방식**: 벌목 시 나무가 완전히 사라졌다가(Scale 0), 매일 아침 조금씩 커지는 방식(0.3 -> 0.6 -> 1.0).
- **트리거**: `TimeManager`에서 발행하는 `DayStartedEvent`를 구독하여 하루 단위로 성장.
- **물리 처리**: 성장이 완료되기 전까지는 콜라이더를 비활성화하여 플레이어가 밟고 지나갈 수 있도록 처리.
- **시각 연출**: 성장이 일어날 때 코루틴을 사용하여 부드럽게 커지는 효과(Lerp) 적용.

---

## 2. 구현해야 할 작업 목록

### [ ] 이벤트 시스템 연동
- [ ] `DayStartedEvent` 구독 및 해제 로직 추가.
- [ ] 하루가 지날 때마다 실행될 `OnDayStarted` 메서드 구현.

### [ ] 성장 로직 구현
- [ ] `_growthStage` (현재 성장 단계) 변수 추가.
- [ ] `_maxGrowthStage` (완전 성장까지 필요한 일수) 설정.
- [ ] 벌목 시 `localScale`을 `Vector3.zero`로 변경.

### [ ] 시각 및 물리 제어
- [ ] 단계별 목표 스케일 계산 로직.
- [ ] 부드러운 성장을 위한 `GrowSmoothly` 코루틴 구현.
- [ ] 단계에 따른 `Collider2D` 활성화/비활성화 제어.

## 3. 핵심 로직 (Core Logic)

### A. 벌목 시 처리 (`FallTree`)
나무가 베어지는 순간 시각적으로 사라지게 만듭니다.
```csharp
private void FallTree()
{
    _isChopped = true;
    _growthStage = 0;
    transform.localScale = Vector3.zero; // 시각적 삭제
    _spriteRenderer.enabled = true; // 객체는 살아있어야 하므로 렌더러는 켬
    
    foreach (var col in _colliders) col.enabled = false; // 플레이어 통과 가능
    
    // 기존 리스폰 코루틴 대신 이벤트 기반으로 변경 준비
}
```

### B. 하루 단위 성장 (`OnDayStarted`)
이벤트를 수신하여 성장 단계를 올리고 스케일을 조절합니다.
```csharp
private void OnDayStarted(DayStartedEvent evt)
{
    if (!_isChopped) return;

    _growthStage++;
    float targetScale = (float)_growthStage / _maxGrowthStage;
    
    // 부드럽게 커지는 연출 시작
    StartCoroutine(GrowSmoothly(targetScale));

    // 성장이 완료되면 복구
    if (_growthStage >= _maxGrowthStage)
    {
        Respawn();
    }
}
```

### C. 부드러운 스케일 애니메이션 (`GrowSmoothly`)
```csharp
private IEnumerator GrowSmoothly(float targetScale)
{
    Vector3 startScale = transform.localScale;
    Vector3 endScale = Vector3.one * targetScale;
    float elapsed = 0f;
    float duration = 1.5f; // 1.5초 동안 성장

    while (elapsed < duration)
    {
        elapsed += Time.deltaTime;
        transform.localScale = Vector3.Lerp(startScale, endScale, elapsed / duration);
        yield return null;
    }
}
```

### D. 완전 복구 (`Respawn`)
```csharp
private void Respawn()
{
    _isChopped = false;
    _currentHits = 0;
    transform.localScale = Vector3.one;
    foreach (var col in _colliders) col.enabled = true; // 다시 벨 수 있게 충돌 활성화
    
    // 반짝이는 이펙트 재생 (선택)
    if (_poolManager != null) _poolManager.Spawn("Sparkle", transform.position, Quaternion.identity);
}
```

## 4. 기대 효과
1. **에셋 절약**: 추가적인 묘목이나 그루터기 스프라이트 없이도 자연스러운 성장 연출 가능.
2. **성능 최적화**: 매 프레임 체크하는 타이머 대신 하루 단위 이벤트를 사용하므로 CPU 부하 최소화.
3. **게임성 향상**: 매일 아침 농장을 둘러보며 자원이 회복되는 것을 시각적으로 확인하는 즐거움 제공.
