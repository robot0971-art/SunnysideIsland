## 7. 수정 권장 사항

### 7.1 높은 우선순위
1. **시작 아이템 추가**: 기본 도끼, 곡괭이 제공 권장
2. **보스 밸런스**: 고블린 족장 체력/데미지 플레이테스트 필요
3. **호박 수익**: 성장 시간 대비 수익이 낮음 (판매가 60G로 상향 권장)

### 7.2 중간 우선순위
1. **허기 감소 속도**: 게임플레이 테스트 후 조정
2. **관광객 수**: 챕터별 자연스러운 증가 확인
3. **퀘스트 진행**: 메인 퀘스트 라인 연속성 확인

### 7.3 낮은 우선순위
1. **가격 미세 조정**: 플레이어 피드백 후 조정
2. **적 스폰율**: 난이도 느낌에 따라 조정
3. **UI 배치**: 사용성 테스트 후 조정
