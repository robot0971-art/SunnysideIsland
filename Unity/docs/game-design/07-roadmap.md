## 7.5 테스트 계획

### 주간 플레이 테스트

- **빈도**: 매주 금요일
- **참여**: 전체 팀
- **목표**: 해당 Phase 기능 테스트
- **산출물**: 버그 리스트, 피드백 문서

### 알파 테스트 (Week 17)

- **대상**: 외부 테스터 5~10명
- **기간**: 1주일
- **목표**: 실제 플레이어 피드백
- **산출물**: 피드백 리포트, 수정 리스트

### 베타 테스트 (Week 18)

- **대상**: 더 넓은 외부 그룹
- **기간**: 3~5일
- **목표**: 최종 버그 발견
- **산출물**: 릴리즈 Go/No-Go 결정
