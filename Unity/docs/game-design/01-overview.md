# 1. 게임 개요

## 1.1 기본 정보

| 항목 | 내용 |
|------|------|
| 게임명 | 선샤인 아일랜드 (Sunnyside Island) |
| 장르 | 생존 / 건설 / 팜딩 / 액션 RPG |
| 타겟 플랫폼 | PC (Windows/Mac) |
| 엔진 | Unity 2022.3 LTS |
| 플레이 시간 | 2.5 ~ 3.5시간 |
| 플레이어 수 | 1인 (싱글플레이어) |
| 시점 | 탑다운 2D (16x16 픽셀 아트) |
| ESRB 등급 | E10+ (10세 이상) |

## 1.2 한 줄 컨셉

버려진 섬에 불시착한 주인공이 자원을 채집하고, 농사와 낚시, 사냥으로 생존하며, 점차 섬을 관광 명소로 발전시키는 생존 건설 시뮬레이션

## 1.3 핵심 가치

- **성취감**: 텐트 하나에서 시작해 번화한 리조트로 발전
- **자유도**: 플레이어의 선택에 따른 다양한 발전 방식
- **힐링**: 귀여운 픽셀 아트와 여유로운 게임플레이
- **도전**: 전투와 생존의 긴장감

## 1.4 플레이어 여정

```
[Day 1-3]  생존자     → 텐트, 낚시, 기본 도구
[Day 4-7]  정착민     → 오두막, 농사, 첫 주민
[Day 8-14] 마을 주민  → 상점, 고블린 교역, 관광객
[Day 15-28] 리조트 주인 → 고급 시설, 축제, 번영
```

## 1.5 게임 흐름 다이어그램

```
자원 채집 (벌목/채광/사냥/낚시)
        ↓
생존 유지 (허기 관리/요리)
        ↓
건설/확장 (농장/상점/주거지)
        ↓
주민/관광객 유치
        ↓
수익 창출 → 더 큰 시설
        ↓
리조트 완성 (엔딩)
```

## 1.6 핵심 시스템 요약

### 생존 시스템
- **허기**: 0-100, 시간이 지나며 감소, 음식으로 회복
- **체력**: 전투/낙하 등으로 감소, 휴식으로 회복
- **스태미나**: 공격/구르기 소모, 자동 회복

### 생산 시스템
- **농사**: 11종 작물, 5단계 성장
- **낚시**: 타이밍 미니게임, 8종 물고기
- **사냥**: 토끼/사슴/멧돼지 등 동물 사냥
- **채집**: 벌목, 채광, 채취

### 전투 시스템
- **전투**: 4방향 공격, 회피(구르기)
- **무기**: 검/도끼/활 등 다양한 무기
- **몬스터**: 고블린, 스켈레톤, 보스

### 건설 시스템
- **건설**: 타일 단위 건설
- **시설**: 주거/상업/관광/생산 시설
- **주민**: NPC 유치 및 고용

### 경제 시스템
- **통화**: 코인(ⓒ)
- **수익**: 판매, 상점, 관광객
- **관광**: 관광객 유치 및 만족도 관리
