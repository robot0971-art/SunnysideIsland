# Unity-Node.js 서버 연동 가이드

이 가이드는 Windows 환경에서 Node.js를 설치하고, 서버를 실행한 후 Unity에서 테스트하는 방법을 순서대로 설명합니다.

---

## 1단계: Node.js 설치

### 1-1. Node.js 다운로드
1. 웹 브라우저를 열고 주소창에 다음을 입력합니다:
   ```
   https://nodejs.org
   ```
2. 웹사이트가 열리면 **"LTS"** (Long Term Support) 버전의 다운로드 버튼을 클릭합니다.
   - LTS는 "장기 지원 버전"으로, 안정적이고 오랫동안 지원됩니다.

### 1-2. Node.js 설치
1. 다운로드한 설치 파일(`node-vxx.x.x-x64.msi`)을 더블클릭하여 실행합니다.
2. 설치 마법사가 열리면 **"Next"** 버튼을 계속 클릭합니다.
3. 라이선스 동의 화면에서는 **"I accept"**를 선택하고 **"Next"**를 클릭합니다.
4. 설치 경로는 기본값(`C:\Program Files\nodejs\`)을 사용하고 **"Next"**를 클릭합니다.
5. **"Install"** 버튼을 클릭하여 설치를 시작합니다.
6. 설치가 완료되면 **"Finish"** 버튼을 클릭합니다.

### 1-3. 설치 확인
1. 키보드에서 **Windows 키 + R**을 동시에 누릅니다.
2. 실행 창이 열리면 `cmd`를 입력하고 **"확인"**을 클릭합니다.
3. 검은색 명령 프롬프트 창이 열립니다.
4. 다음 명령어를 입력하고 Enter를 누릅니다:
   ```cmd
   node --version
   ```
5. 버전 번호(예: `v20.x.x`)가 표시되면 설치가 성공적으로 완료된 것입니다.

---

## 2단계: 서버 파일 준비

### 2-1. 프로젝트 폴더 만들기
1. 바탕화면이나 원하는 위치에 새 폴더를 만듭니다.
2. 폴더 이름을 `unity-server`로 지정합니다.

### 2-2. 서버 파일 저장
1. 제공된 `server.js` 파일을 `unity-server` 폴더 안에 복사합니다.
2. 파일이 제대로 복사되었는지 확인합니다.

---

## 3단계: 서버 실행

### 3-1. 명령 프롬프트에서 폴더 이동
1. 다시 **Windows 키 + R**을 누르고 `cmd`를 입력하여 명령 프롬프트를 엽니다.
2. 서버 폴더로 이동합니다. 예를 들어 바탕화면에 폴더를 만들었다면:
   ```cmd
   cd Desktop\unity-server
   ```
   - 참고: `cd`는 "change directory(폴더 변경)"의 약자입니다.

### 3-2. 서버 실행
1. 다음 명령어를 입력하고 Enter를 누릅니다:
   ```cmd
   node server.js
   ```
2. 다음과 같은 메시지가 표시되면 서버가 정상적으로 실행 중입니다:
   ```
   ========================================
   서버가 http://localhost:3000 에서 실행 중입니다.
   유니티에서 데이터를 보내 여기에 표시됩니다.
   ========================================
   
   서버를 종료하려면 Ctrl+C를 누르세요.
   ```

### 3-3. 서버 종료 방법
- 서버를 종료하려면 명령 프롬프트 창에서 **Ctrl + C** 키를 동시에 누르고, 종료 확인 메시지가 나타나면 **Y**를 입력한 후 Enter를 누릅니다.

## 4단계: Unity에서 테스트

### 4-1. Unity 프로젝트 준비
1. Unity Hub를 열고 새 프로젝트를 생성하거나 기존 프로젝트를 엽니다.

### 4-2. 스크립트 추가
1. `UnityScoreSender.cs` 파일을 Unity 프로젝트의 `Assets` 폴더에 복사합니다.
   - `Assets` → `Scripts` 폴더 안에 넣는 것을 권장합니다.

### 4-3. 씬 설정
1. **Hierarchy** 창에서 **우클릭** → **UI** → **Button**을 선택하여 버튼을 생성합니다.
   - 버튼이 자동으로 Canvas와 EventSystem도 함께 생성됩니다.
2. **Hierarchy** 창에서 **우클릭** → **Create Empty**를 선택하여 빈 오브젝트를 생성합니다.
3. 빈 오브젝트의 이름을 `ScoreManager`로 변경합니다.

### 4-4. 스크립트 연결
1. `ScoreManager` 오브젝트를 선택합니다.
2. **Inspector** 창에서 **"Add Component"** 버튼을 클릭합니다.
3. `Unity Score Sender`를 검색하거나 **Scripts** 폴더에서 찾아 추가합니다.
4. **Inspector**에서 다음 설정을 확인합니다:
   - **Server Url**: `http://localhost:3000` (기본값 유지)
   - **Player Name**: 원하는 이름 입력 (예: "테스트플레이어")
   - **Player Score**: 원하는 점수 입력 (예: 100)
   - **Send Button**: Hierarchy의 Button을 드래그하여 연결

### 4-5. 테스트 실행
1. Unity 에디터의 **Play** 버튼(▶)을 클릭하여 게임을 실행합니다.
2. Game 뷰에서 버튼을 클릭합니다.
3. Node.js 서버가 실행 중인 명령 프롬프트 창을 확인합니다.
4. 다음과 같은 메시지가 표시되면 성공입니다:
   ```
   테스트플레이어님이 100점을 달성했습니다!
   ```

## 5단계: 문제 해결

### 서버가 시작되지 않는 경우
- **원인**: 이미 3000번 포트를 사용하는 다른 프로그램이 있을 수 있습니다.
- **해결 방법**: 
  - 다른 프로그램을 종료하거나
  - `server.js` 파일에서 `const PORT = 3000;`을 다른 숫자(예: 3001)로 변경

### Unity에서 연결 실패
- **원인 1**: 서버가 실행 중이지 않습니다.
  - **해결 방법**: 명령 프롬프트에서 `node server.js`를 다시 실행합니다.
  
- **원인 2**: 방화벽이 차단합니다.
  - **해결 방법**: Windows 방화벽에서 Node.js를 허용하거나, 잠시 방화벽을 끕니다.

- **원인 3**: URL이 잘못되었습니다.
  - **해결 방법**: `UnityScoreSender.cs`의 Server Url이 `http://localhost:3000`인지 확인합니다.

### JSON 데이터가 제대로 전송되지 않는 경우
- Unity 콘솔 창(Window → General → Console)에서 에러 메시지를 확인합니다.
- `UnityScoreSender.cs`의 `playerName`과 `playerScore`에 값이 제대로 설정되었는지 확인합니다.

## 요약

1. **Node.js 설치**: https://nodejs.org 에서 LTS 버전 다운로드 및 설치
2. **서버 준비**: `server.js` 파일을 원하는 폴더에 저장
3. **서버 실행**: 명령 프롬프트에서 `node server.js` 실행
4. **Unity 설정**: `UnityScoreSender.cs`를 추가하고 버튼에 연결
5. **테스트**: Unity Play 모드에서 버튼 클릭

---

## 참고사항

- 서버와 Unity는 **같은 컴퓨터**에서 실행해야 합니다.
- 서버는 항상 **Unity보다 먼저** 실행되어야 합니다.
- 서버를 종료하지 않으면 계속 실행 상태로 유지됩니다.
